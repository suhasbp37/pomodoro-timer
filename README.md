# Pomodoro Timer ⏱️

A simple, clean Pomodoro timer web app with task management.

## Features
- ✅ 25 min Pomodoro, 5 min Short Break, 15 min Long Break
- ✅ Start, Pause, Reset controls
- ✅ Add, complete, delete tasks
- ✅ Auto-switch modes
- 🔔 Audio notification when timer ends
- 💾 Tasks saved to localStorage
- 🎨 Modern red theme (Pomofocus style)

## Tech Stack
- **Frontend:** HTML, CSS, Vanilla JavaScript
- **Backend:** Node.js with Express
- **Storage:** localStorage + JSON

## Setup & Run

```bash
# Install dependencies
npm install

# Start server
npm start
